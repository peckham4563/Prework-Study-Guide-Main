HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Common elements used:
  • "h1,2,3,...": Heading
  • "p" -Paragraph
  • "ul" -Unordered List
  • "ol" -Ordered List
  • "li" -List
  • "img" -Image
  • "br" -Line Break
  • "a" -Anchor
  • "src" -Source
  • "alt" -Describes appearance and funtionality.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS follows a pattern of an initial selector, followed by your declaration block of declarations that is made up of a property and a value.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • "cd" -Current Directory
  • "ls" -List
  • Git status displays the state of the working directory
  • Git push sends commits and asks them to update and git pullis used to sync remote content

JavaScript

  • End lines of JavaScript with a semicolon
  • All strings are surrounded with quotations marks
  • We define topics AFTER we tell the computer to log it into the console.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
